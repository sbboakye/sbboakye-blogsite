## Installation

### Prerequisites

- Scala 3.5.0
- SBT (Scala Build Tool) 1.x or later
- Java 11 or later

### Steps

1. Clone the repository:

    ```bash
    git clone https://github.com/username/project-name.git
    ```

2. Navigate to the project directory:

    ```bash
    cd project-name
    ```

3. Compile the project:

    ```bash
    sbt compile
    ```

4. Run the project:

    ```bash
    sbt run
    ```

5. Run tests:

    ```bash
    sbt test
    ```
